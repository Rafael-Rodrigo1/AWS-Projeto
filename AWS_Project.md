# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 01/01/2026
Empresa: Abstergo Industries 
Responsável: Rafael Rodrigo Vital Santos

## Introdução
Este relatório apresenta o processo de Implementação de ferramentas na empresa Abstergo Industries, realizado por Rafael Rodrigo Vital Santos. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de Implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon S3
- Armazenamento de arquivos e documentos na nuvem
- A empresa farmacêutica pode utilizar o Amazon S3 para armazenar contratos, relatórios financeiros,
  notas fiscais, documentos regulatórios e arquivos trocados com outras empresas parceiras.
  Isso elimina custos com servidores físicos, manutenção e risco de perda de dados.

Etapa 2: 
- Amazon EC2
- Servidores virtuais sob demanda
-  Amazon EC2 pode ser utilizado para hospedar sistemas internos da empresa,
  como um sistema de gestão financeira, integração com parceiros comerciais
  ou APIs para troca de informações entre empresas.

Etapa 3: 
- AWS Lambda
- Execução de código sem necessidade de servidor
- A empresa pode usar o AWS Lambda para automatizar processos simples,
  como envio de relatórios financeiros, validação de dados recebidos de parceiros
  ou integração automática entre sistemas de diferentes empresas.



## Conclusão
A Implementação de ferramentas na empresa *Abstergo Industries tem como esperado Redução significativa de custos operacionais,Maior segurança e confiabilidade dos dados e Facilidade de integração com outras empresas*, o que aumentarão a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilizaçâo das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
